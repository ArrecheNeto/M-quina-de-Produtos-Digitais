# arrecheneto-produtos

Marketplace do plugin **Máquina de Produtos Digitais**, de Helio Arreche.

## Instalar no Claude Cowork

Adicione este marketplace e instale o plugin `maquina-de-produtos-digitais`. Depois rode **/setup** e **/maquina**.

## O plugin

**Máquina de Produtos Digitais** — esteira de 7 passos que transforma o que você já sabe num produto digital pronto pra vender, sem API e sem custo, na sua própria assinatura do Claude. Pesquisa dores reais, escolhe a promessa, monta o produto mínimo, gera a página de vendas, mapeia 10 compradores, cria o lançamento e analisa a validação — tudo num painel local.

Documentação completa em [`maquina-de-produtos-digitais/README.md`](./maquina-de-produtos-digitais/README.md).
